# Структура файлов данных из dataset

#### movies.csv

* `movieId` — ID

* `title` — название

* `genres` — жанры

#### ratings.csv

* `userId` — ID пользователя

* `movieId` — ID фильма

* `rating` — оценка

* `timestamp` — временная метка

#### users.txt

* ID

* имя

* почта

* пол

* дата регистрации

* профессия

Без строки заголовков

#### genres.txt

* жанр

Без строки заголовков

#### occupation.txt

* профессия

Без строки заголовков

### tags.csv

— `userId` — ID пользователя

— `movieId` — ID фильма  

— `tag` — тег

— `timestamp` — временная метка
